# Ego Tracker 📊

A sleek, modern progress tracker designed specifically for **Semester 4 subjects**. Track your study progress across multiple courses, units, and subtopics with an intuitive interface and real-time analytics.

**Live Demo:** [ego-tracker.vercel.app](https://ego-tracker.vercel.app)

---

## 🎯 Features

- **Multi-Course Tracking**: Manage progress across multiple courses (LANT, OOP, DBMS, etc.)
- **Unit-Based Organization**: Break down each course into manageable units
- **Subtopic Management**: Add, track, and monitor individual subtopics within units
- **Progress Visualization**: 
  - Real-time percentage calculations
  - Visual progress bars for each unit
  - Chart-based breakdown of completion status
- **Status Indicators**: Track each subtopic as:
  - 🔴 **Not Started** (0%)
  - 🟠 **In Progress** (50%)
  - 🟢 **Done** (100%)
- **Local Persistence**: All data is saved to browser localStorage
- **Dark Mode UI**: Modern, eye-friendly dark interface with lime accent color
- **Responsive Design**: Works seamlessly on desktop and mobile devices
- **Motivational Feedback**: Get personalized roasts based on your progress! 😄

---

## 🚀 Quick Start

### Prerequisites
- Any modern web browser (Chrome, Firefox, Safari, Edge)
- No installation or build process required

### Usage

1. **Visit the app**: Go to [ego-tracker.vercel.app](https://ego-tracker.vercel.app)
2. **Select a Course**: Click on a course tab (e.g., LANT, OOP, DBMS)
3. **Choose a Unit**: Select a unit within the course
4. **Add Subtopics**: Type subtopic names and click "+ ADD"
5. **Track Progress**: Click on subtopics to cycle through status:
   - Not Started → In Progress → Done → Not Started (repeat)
6. **Monitor Analytics**: View real-time progress metrics and charts

### Features Explained

#### Dashboard Metrics
- **Overall %**: Your cumulative progress across all courses
- **Done**: Number of completed subtopics
- **In Progress**: Number of subtopics being studied
- **Total**: Overall number of subtopics added

#### Unit Breakdown Chart
Visual representation of each unit's completion status using:
- 🟢 Green: Done
- 🟠 Orange: In Progress
- 🔴 Red: Not Started

🎮 Tips & Tricks
Keyboard Shortcut: Press Enter to quickly add a subtopic
Sort by Status: Hover over subtopics to see detailed borders
Progress Roasts: The motivational messages change based on your progress
Cross-Course View: Switch courses instantly to compare progress
🐛 Known Limitations
Data is stored locally (not synced across devices)
Clearing browser cache will reset all progress
Does not support collaborative tracking
